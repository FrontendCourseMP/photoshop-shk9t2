# Редактор изображений

Веб-приложение для работы с изображениями, разработанное в рамках курса по цифровой обработке изображений.

## Демо

https://starlit-pika-08a601.netlify.app/

## Лабораторные работы

### Лабораторная работа 1 — Цифровое представление изображения

- Загрузка изображений в форматах PNG, JPG и GB7
- Отображение изображения на холсте HTML5 Canvas
- Сохранение изображения в форматах PNG, JPG и GB7 с редактируемым именем файла
- Собственная реализация кодека GrayBit-7 — декодер и кодер с нуля
- Строка состояния с шириной, высотой, глубиной цвета и масштабом
- Автоматическое масштабирование при загрузке (отступы 50px)
- Масштабирование колёсиком мыши от 12% до 300%
- Адаптация к изменению размера окна через `ResizeObserver`
- Шахматный фон для отображения прозрачности

### Лабораторная работа 2 — Цветовые каналы и инструмент «Пипетка»

- Панель каналов с миниатюрами R, G, B и Alpha
- Включение и выключение каналов с мгновенным отражением на холсте
- Оригинальные данные изображения не изменяются — все операции на копии
- Инструмент «Пипетка» — активируется кнопкой в панели инструментов
- Отображение координат X/Y пикселя, значений R/G/B/A и HEX
- Конвертация цвета в пространство CIELAB через XYZ

### Лабораторная работа 3 — Градационные преобразования и «Уровни»

- Инструмент «Уровни» через диалоговое окно `<dialog>`
- Гистограмма с линейной и логарифмической шкалой
- Настройка для всех каналов одновременно (Master) или отдельно R/G/B/A
- Три маркера: чёрная точка, гамма (0.1–9.9), белая точка
- Маркеры не пересекаются, гамма всегда находится между маркерами
- Таблицы подстановки (LUT) для быстрой обработки
- Предпросмотр в реальном времени с троттлингом
- Кнопки: Сбросить, Отмена, Применить

### Лабораторная работа 4 — Масштабирование изображения

- Собственная реализация метода ближайшего соседа
- Собственная реализация билинейной интерполяции (по умолчанию)
- Реестр методов для лёгкого добавления новых алгоритмов
- Слайдер масштаба 12%–300% в строке состояния
- Диалог изменения размера с единицами `px` и `%`
- Информация о размере до и после в мегапикселях
- Чекбокс сохранения пропорций, тултип с описанием алгоритма, валидация вводимых значений

### Лабораторная работа 5 — Фильтрация изображений

- Фильтрация через свёртку с ядром 3×3
- Шесть предустановленных ядер: тождественное отображение, повышение резкости, фильтр Гаусса, прямоугольное размытие, операторы Прюитта X и Y
- Редактируемая сетка 3×3 для ручного ввода ядра
- Применение к выбранным каналам R/G/B/A независимо
- Три стратегии обработки краёв: заполнение чёрным, белым, копирование
- Обработка через Web Worker — интерфейс не блокируется
- Предпросмотр в реальном времени
- Кнопки: Сбросить, Отмена, Применить

## Формат GrayBit-7 (GB7)

Учебный формат изображений в оттенках серого без сжатия. Использует 7 бит на пиксель для значения яркости, 8-й бит — опциональная маска прозрачности.

Структура файла:

- 4 байта сигнатуры
- 8 байт заголовка
- затем данные пикселей

Декодер и кодер реализованы с нуля.
